# 上传

> `git add .`
> `git commit -m '需要说的话'`//本地仓库
> `git push`//线上仓库
> `git checkout 分支名`//进入分支，一般是在分支下面进入`master`主分支
> `git merge origin/分支名` 把分支名合并到`merge`
> `git push`//上传线上仓库
> `git pull`//创建分支之后要在本地拉取下来
> `git status`//查看未同步的代码
> `git checkout .`//清除这次更改
> `git branch`//查看分支合并情况
> `git merge 分支名`//分支融合，也就是新分支和以前哪个分支进行合并
