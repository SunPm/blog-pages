### includes  返回值布尔  true/ false

> 判断我们字符串中有没有指定字符

```js
// includes('指定字符',开始查找的位置(可选，默认 0 ))
// 第二个参数是一个数字，不是数字默认转为数字
let str = '12345'
console.log(str.includes('1')) // true
console.log(str.includes('1',2)) // false
console.log(str.includes('6')) // false
```
