# state

> `state`是一种状态对象，状态对象是在做单页应用的时候，里面的共享值，就叫做状态 

## 声明状态

```js
//store/index.js
const state = {
  count: 3
}
```

## 在页面访问状态对象方法

1. 计算属性

    ```js
    computed:{
        count(){
            return this.$store.state.count
        }
    }
    ```



2. `mapState`

   ```js
   import { mapState } from 'vuex'
   computed: mapState({
       count: state => state.count
   })
   ```



3. `mapState`数组的形式(**推荐**)

   ```js
   import { mapState } from 'vuex'// mapState -> 引用状态
   computed: mapState(['count'])
   ```
